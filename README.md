# standingquestions.com

Source for [standingquestions.com](https://standingquestions.com) — the home of the **Standing Questions** pattern (store the questions, re-derive the answers).

Plain hand-written HTML + CSS, served by GitHub Pages. No framework, no build step, no JavaScript.

The pattern itself — schemas, seed questions, Claude Code skill, and metrics — lives at [Rocco-alt/standing-questions](https://github.com/Rocco-alt/standing-questions).

By [Dr Raj Kadiwar](https://www.linkedin.com/in/raj-kadiwar/) and [Krishna Kadiwar](https://www.linkedin.com/in/krishnajk/). MIT.
